Detailed Description

Deep-sleep orchestration and wake-up cause handling.

Owns the station duty cycle: classifies why the MCU woke up, keeps the boot counter in RTC RAM up to date and puts the system back into deep sleep with the timer (and, from Increment 4, the rain-gauge EXT wake-up) armed.

Usage example:

PowerManager power;
power.begin(); // classify wake-up, ++bootCount
Serial.println(power.wakeupCauseString());
power.deepSleep(config.station.wakeIntervalS); // never returns

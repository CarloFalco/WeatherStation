◆ triggerConversion()

bool Ina3221Driver::triggerConversion ( )

Start one single-shot conversion of all three channels.

Configuration: 16-sample averaging, 1.1 ms conversion time for both bus and shunt (~106 ms total). The chip returns to power-down by itself when the conversion set completes.

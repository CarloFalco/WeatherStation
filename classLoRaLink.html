bool LoRaLink::receiveRaw ( uint8_t *  buf,
size_t  bufSize,
size_t &  len,
uint32_t  timeoutMs 
)

Listen for one raw binary packet (OTA chunks).

LoRa-specific (not part of ITelemetryLink): the OTA transfer uses raw frames because base64-in-JSON would waste ~33% of the scarce 255-byte LoRa payload.

Parameters
bufDestination buffer; MUST hold a full LoRa packet (>= 256 B).
bufSizeSize of buf.
lenSet to the received packet length on success.
timeoutMsMaximum listening time [ms].
Returns
true if a packet passed CRC and fit the buffer.

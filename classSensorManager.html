Detailed Description

Owns the list of registered sensors and drives their life cycle.

Sensors are registered once with add(); every wake-up cycle main.cpp calls beginAll() and then readAll(). A sensor whose begin() fails is marked unhealthy and skipped for the rest of the cycle, so one broken sensor never blocks the others: its JSON fields are simply absent.

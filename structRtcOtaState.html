Detailed Description

OTA transfer progress, preserved across deep sleep.

A full firmware image takes ~20 minutes over LoRa: without this, any interruption would restart the transfer from chunk 0. The station resumes from nextChunk as soon as the base offers the same image again (matching size/crc/chunks).
